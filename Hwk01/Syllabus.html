Catalog Description

Basic components of web development which include aspects of design as well as current
development technologies. Development technologies include, but are not limited to,
HTML/XHTML, JavaScript, and CSS. Other technologies discussed may include Java
Applets, CGI programming, ASP.NET and PHP.

Prerequisites: CST 211 or CST 313 or instructor’s consent.

Objectives

Upon completion of this course students should be able to:

  1. Design and implement web sites using HTML 5 and CSS3
  2. Optimize site resources with tools available through NodeJS
  3. Work with JavaScript frameworks such as jQuery and Angular
  4. Integrate web services for a dynamic user experience
  5. Create modern and maintainable web sites.
  6. Build a functional website based on a designer-provided sketch

Grading

The final grade will be determined from the following breakdown:
Assignments, Quizzes, and Labs 60%; Mid-Term 20%; Final 20%

The grade calculations will be as follows:
90% + -> A; 80% + -> B; 70% + -> C; 60% + -> D; Less than 60% -> F

Assignments

Assignments are typically web pages or that generally deal with implementing a specific aspect
within web development. The due date of all assignments will be announced when they are
assigned. Late assignments will be given a 20% penalty for each day it is late.

Tests/Quizzes

There will be a mid-term and a final to demonstrate that you have learned how to use web
technologies, and also understand the reasoning behind certain structure/design decisions. Any
quizzes will be announced at least a week in advance.
